# Data-Cleaning-and-Dashboard-in-Excel
📊 Bike Sales Dashboard (Excel)
📌 Project Overview
This project focuses on cleaning, analyzing, and visualizing bike sales data using Microsoft Excel. The raw dataset was first cleaned and transformed in Excel, then analyzed using Pivot Tables, charts, and slicers to build an interactive dashboard that highlights patterns in customer bike purchases.
Dashboards combine multiple visual elements such as charts and tables to provide an at-a-glance view of important data and trends, helping users quickly understand insights from large datasets.

🎯 Project Objective
The objective of this project was to analyze customer demographics and behavior to understand the factors that influence bike purchases and present those insights through a clear and interactive Excel dashboard.

🛠️ Tools & Techniques Used
* Microsoft Excel
* Data Cleaning
* Pivot Tables
* Data Transformation
* Data Visualization
* Interactive Dashboard Design

⚙️ Project Workflow
1️⃣ Data Cleaning
-Removed duplicates
-Standardized categorical values
-Corrected data formatting
-Created new fields such as Age Brackets for better analysis
2️⃣ Data Analysis
-Used Pivot Tables to summarize and analyze the dataset.
3️⃣ Data Visualization
-Built charts to analyze:
-Average income per bike purchase
-Customer age groups
-Commute distance vs bike purchases
4️⃣ Dashboard Creation
Designed an interactive Excel dashboard with slicers to filter by:
-Marital Status
-Education
-Region

📊 Dashboard Insights
-The dashboard helps identify:
-Which age groups are more likely to purchase bikes
-The relationship between income and bike purchases
-How commute distance affects buying behavior
-Differences across regions and education levels
